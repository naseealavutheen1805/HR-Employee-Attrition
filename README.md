# Adecco HR Employee Attrition Analysis Dashboard

## Overview
This project analyzes employee attrition at Adecco India. The dataset was cleaned, explored using pivot tables, and visualized in an Excel dashboard to understand which employee groups are more likely to leave and what factors contribute to attrition.

## Key Metrics
- **Total Employees:** ~1,470  
- **Average Tenure:** 7 years  
- **Attrition Rate:** 16.12%  
- **Average Monthly Income:** ~$6503  

## What I Did
- Cleaned and formatted the HR dataset  
- Answered case study questions using Pivot Table analysis  
- Analyzed attrition by department, job role, age, salary, ratings, and marital status  
- Built an interactive Excel dashboard with slicers for Department, Gender, Age, Job Satisfaction, Attrition, and Work-Life Balance  

## Insights
- Sales and Human Resources show higher attrition levels  
- Employees aged **25–35** and **42–49** show notable attrition patterns  
- Lower performance ratings correlate with higher attrition  
- Monthly income and age both influence attrition trends  
- Certain job roles (Sales Rep, Sales Executive) show higher churn  

## Deliverables
- Cleaned Dataset  
- Pivot Table Analysis / Solution Guide  
- HR Attrition Dashboard (Excel)  
- Project Document  

## Conclusion
This analysis highlights the major factors driving employee turnover and supports HR decisions related to retention, workload balance, and employee engagement.
